# 归档版本说明（可作 GitHub Release 描述）

本仓库 **MAIBOT（CCB）** 已进入**归档状态**，以**参考代码**形式**公开发布**，供社区学习、借鉴与合作交流。

## 本版本包含

- **技术栈**：LangChain / LangGraph / DeepAgent + FastAPI + Electron + React（@assistant-ui）
- **功能范围**：AI 对话、工作台、任务管线、Skills、知识库与可插拔插件等完整实现
- **文档**：架构、运维、契约与历史过程文档位于 `docs/` 与 `docs/archive/`

## 项目优势（摘要）

- **主流生态**：与 LangChain/LangGraph/DeepAgent 官方及社区实践对齐，便于迁移与二次开发  
- **全栈可运行**：后端 + Electron 前端完整打通，可直接克隆运行体验  
- **生产级设计**：SQLite 持久化、向量懒加载、中间件链与门禁脚本齐全  
- **文档与契约**：架构、管线、Skills/知识库等文档集中，推荐阅读路径清晰  

完整优势与适用场景见 [docs/ADVANTAGES.md](docs/ADVANTAGES.md)。

## 适用对象

- 希望参考 **LangGraph + FastAPI + Electron** 全栈 AI 应用结构的开发者  
- 需要了解 **DeepAgent、中间件、Skills、知识库** 等模块划分与契约的架构设计者  
- 学习 **流式对话、会话状态、工作区与任务管线** 等前后端联调实现
