# check-homework

Use this command to evaluate homework and return clear, actionable student feedback.

## Instructions

1. Use `homework-check` to score work against provided rubric and requirements.
2. Use `student-learning` to tailor feedback to the student's current mastery level.
3. Highlight strengths, mistakes, and concrete improvement steps.
4. Return concise grading summary, evidence notes, and next-study recommendations.
