# Productivity Plugin

This plugin provides workflow-first tools for daily execution and personal/team productivity:

- Build daily dashboards for progress and blockers
- Maintain memory hygiene for decisions and follow-ups
- Manage tasks with clear priorities and delivery rhythm

## Structure

- `commands/`: task entrypoints for common productivity requests
- `skills/`: Claude-style skills for daily workflows
- `.claude-plugin/plugin.json`: plugin manifest
- `.mcp.json`: MCP connector registry placeholder
- `CONNECTORS.md`: connector setup notes
