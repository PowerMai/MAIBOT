# triage

Use this command to process and prioritize incoming support tickets.

## Instructions

1. Use `ticket-triage` to classify tickets by urgency, impact, and category.
2. Use `customer-research` to add account context and history signals.
3. Use `response-drafting` to prepare first response drafts.
4. Use `knowledge-management` to link reusable solution articles.
5. Return priority queue, response drafts, and assignment recommendations.
