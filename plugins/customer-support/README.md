# Customer Support Plugin

This plugin provides Claude-style support workflows for fast triage and high-quality customer responses.

- Prioritize tickets by urgency and impact
- Draft accurate and empathetic responses
- Manage escalations with clear ownership
- Improve support quality through knowledge reuse

## Structure

- `commands/`: ticket and escalation entrypoints
- `skills/`: reusable support capabilities
- `.claude-plugin/plugin.json`: plugin manifest
- `.mcp.json`: MCP connector registry placeholder
- `CONNECTORS.md`: connector setup guidance
