---
name: create-cowork-plugin
description: Guide users through creating a new plugin from scratch in a cowork session, from discovery to packaging.
compatibility: Requires Cowork desktop app environment with access to the outputs directory for delivering .plugin files.
---

## Workflow

1. Clarify plugin goals, target users, and required components.
2. Define commands, skills, and connector expectations.
3. Produce plugin structure, validate configuration, and package output.
