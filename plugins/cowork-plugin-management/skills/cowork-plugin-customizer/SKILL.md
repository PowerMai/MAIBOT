---
name: cowork-plugin-customizer
description: Customize an existing plugin for organization-specific tools, workflows, and connector configuration.
compatibility: Requires Cowork desktop app environment with access to mounted plugin directories (mnt/.local-plugins, mnt/.plugins).
---

## Workflow

1. Read current plugin structure and locate customization points.
2. Align commands, skills, and connectors to organization standards.
3. Validate updates and package the customized plugin output.
