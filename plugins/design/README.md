# Design Plugin

Claude-style design workflows for accessibility, critique, UX writing, research synthesis, design systems, and developer handoff.

## Structure

- `commands/`: design execution entrypoints
- `skills/`: reusable design capabilities
- `.claude-plugin/plugin.json`: plugin manifest
- `.mcp.json`: connector configuration
- `CONNECTORS.md`: connector setup guidance

## Included Skills

- `accessibility-review`
- `design-critique`
- `design-handoff`
- `design-system-management`
- `user-research`
- `ux-writing`

## Included Commands

- `accessibility`
- `critique`
- `design-system`
- `handoff`
- `research-synthesis`
- `ux-copy`
