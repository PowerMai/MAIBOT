# Bio Research Plugin

Claude-style bio research workflows for scientific problem framing, omics data quality control, and pipeline development.

## Structure

- `commands/`: bio research task entrypoints
- `skills/`: reusable scientific capabilities
- `.claude-plugin/plugin.json`: plugin manifest
- `.mcp.json`: connector configuration
- `CONNECTORS.md`: connector setup guidance

## Included Skills

- `instrument-data-conversion`
- `nextflow-development`
- `scientific-problem-selection`
- `scvi-tools`
- `single-cell-rna-qc`

## Included Commands

- `start`
