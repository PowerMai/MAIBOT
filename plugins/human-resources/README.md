# Human Resources Plugin

Claude-style HR workflows for recruiting, onboarding, compensation, policy guidance, and people analytics.

## Structure

- `commands/`: task entrypoints for common HR operations
- `skills/`: reusable HR capabilities
- `.claude-plugin/plugin.json`: plugin manifest
- `.mcp.json`: connector configuration
- `CONNECTORS.md`: connector setup guidance

## Included Skills

- `compensation-benchmarking`
- `employee-handbook`
- `interview-prep`
- `org-planning`
- `people-analytics`
- `recruiting-pipeline`

## Included Commands

- `comp-analysis`
- `draft-offer`
- `onboarding`
- `people-report`
- `performance-review`
- `policy-lookup`
