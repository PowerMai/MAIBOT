# Marketing Connectors

This plugin is connector-ready. Configure integrations through `.mcp.json`.

Suggested connector categories:

- Ad and campaign platforms
- Web and product analytics tools
- CMS and content operations systems
- Social media and listening platforms

## Setup Notes

1. Register MCP servers in `plugins/marketing/.mcp.json`.
2. Use read-only scopes for analysis and planning first.
3. Enable write scopes for approved publishing and campaign actions.
4. Keep secrets in environment configuration, never in plugin files.
