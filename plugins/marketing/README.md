# Marketing Plugin

This plugin provides Claude-style marketing workflows for strategy, messaging, and execution analytics.

- Define campaign goals and audience strategy
- Keep outputs aligned with brand voice and positioning
- Produce campaign-ready content assets
- Analyze performance and optimization opportunities

## Structure

- `commands/`: campaign and content operation entrypoints
- `skills/`: reusable marketing capabilities
- `.claude-plugin/plugin.json`: plugin manifest
- `.mcp.json`: MCP connector registry placeholder
- `CONNECTORS.md`: connector setup guidance
