# Sales Plugin

This plugin provides Claude-style sales workflows for account development, outbound execution, and bidding lifecycle operations.

- Research target accounts and buying centers
- Prepare call plans with clear goals and objections
- Draft outreach assets tailored to pipeline stage
- Generate daily briefings for sales priorities
- Run bid package reviews and compliance audits
- Draft proposal narratives for bid submission

## Structure

- `commands/`: guided sales execution entrypoints
- `skills/`: reusable sales capabilities
- `.claude-plugin/plugin.json`: plugin manifest
- `.mcp.json`: MCP connector registry placeholder
- `CONNECTORS.md`: connector setup guidance
