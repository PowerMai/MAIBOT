# Engineering Plugin

Streamline engineering workflows — standups, code review, architecture decisions, incident response, and technical documentation.

## What This Plugin Covers

- Standup and execution-status updates
- Code review and risk analysis
- Architecture and system design decisions
- Incident response and postmortem support
- Testing strategy and technical documentation

## Structure

- `commands/`: task-oriented workflow entrypoints
- `skills/`: reusable engineering capabilities
- `.claude-plugin/plugin.json`: plugin manifest
- `.mcp.json`: MCP connector configuration
- `CONNECTORS.md`: connector setup guidance

## Included Commands

- `architecture`
- `debug`
- `deploy-checklist`
- `incident`
- `review`
- `standup`

## Included Skills

- `code-review`
- `documentation`
- `incident-response`

- `tech-debt`
