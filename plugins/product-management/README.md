# Product Management Plugin

This plugin provides Claude-style product workflows for requirement clarity and execution alignment.

- Write clear feature specs with acceptance criteria
- Manage roadmap priorities with measurable impact
- Track product metrics and decision outcomes
- Coordinate stakeholder communication rhythm

## Structure

- `commands/`: spec and roadmap execution entrypoints
- `skills/`: reusable product management capabilities
- `.claude-plugin/plugin.json`: plugin manifest
- `.mcp.json`: MCP connector registry placeholder
- `CONNECTORS.md`: connector setup guidance
