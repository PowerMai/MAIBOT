# Enterprise Search Plugin

This plugin provides Claude-style enterprise search workflows for retrieval planning and knowledge synthesis.

- Design search strategy for ambiguous enterprise questions
- Manage source quality, freshness, and authority
- Synthesize findings into decision-ready outputs

## Structure

- `commands/`: search and governance entrypoints
- `skills/`: reusable enterprise search capabilities
- `.claude-plugin/plugin.json`: plugin manifest
- `.mcp.json`: MCP connector registry placeholder
- `CONNECTORS.md`: connector setup guidance
