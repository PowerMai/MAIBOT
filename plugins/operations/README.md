# Operations Plugin

Claude-style operations workflows for process design, change planning, capacity management, risk control, and vendor operations.

## Structure

- `commands/`: operations workflow entrypoints
- `skills/`: reusable operations capabilities
- `.claude-plugin/plugin.json`: plugin manifest
- `.mcp.json`: connector configuration
- `CONNECTORS.md`: connector setup guidance

## Included Skills

- `change-management`
- `compliance-tracking`
- `process-optimization`
- `resource-planning`
- `risk-assessment`
- `vendor-management`

## Included Commands

- `capacity-plan`
- `change-request`
- `process-doc`
- `runbook`
- `status-report`
- `vendor-review`
