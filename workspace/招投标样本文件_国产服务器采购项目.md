# 国产服务器设备采购招投标文件

## 项目基本信息

**项目名称**: 2025年度国产服务器设备采购项目
**项目编号**: CCB-2025-SERVERS-001
**招标人**: 中国长城计算有限公司
**招标预算**: ¥2,500万元
**投标截止时间**: 2025年1月15日 下午 3:00
**项目周期**: 12个月（2025年3月-2026年2月）

---

## 一、采购需求

### 1.1 需求概述

采购以下国产服务器设备：

| 设备类型 | 数量 | 用途 | 优先级 |
|---------|------|------|-------|
| 鲲鹏服务器 RM2288HA | 50台 | 通用计算 | 必须 |
| 飞腾服务器 FT-2000+ | 30台 | 高性能计算 | 必须 |
| 海光服务器 | 20台 | 备选方案 | 可选 |

### 1.2 技术要求

#### 鲲鹏服务器 RM2288HA
- **CPU**: 鲲鹏 920 处理器（64核）
- **内存**: 最小 256GB DDR4
- **存储**: 最小 4TB SSD + 2TB HDD
- **网络**: 10Gbps 网卡 × 4
- **功耗**: ≤ 800W
- **冷却**: 液冷支持

#### 飞腾服务器 FT-2000+
- **CPU**: 飞腾 FT-2000+ 处理器（64核）
- **内存**: 最小 512GB DDR4
- **存储**: 最小 8TB SSD
- **网络**: 25Gbps 网卡 × 2
- **功耗**: ≤ 1200W
- **虚拟化**: KVM 支持

### 1.3 商务要求

- **交付周期**: 6个月内完成全部交付
- **质保期**: 3年免费维护
- **上门安装**: 包含
- **培训**: 10人次免费培训
- **付款条件**: 30% 预付款，60% 交付时支付，10% 质保期后

## 四、投标文件构成

### 4.1 第一册：资质证明材料

- 营业执照、法人身份证
- ISO9001 质量管理体系认证
- 国产服务器生态认证
- 历年成功案例清单
- 主要技术人员简历

### 4.2 第二册：技术方案 (20-30页)

1. **需求分析** (2-3页)
   - 理解招标需求
   - 需求覆盖矩阵

2. **整体架构** (3-5页)
   - 系统架构图
   - 技术栈选择
   - 创新点说明

3. **具体方案** (8-12页)
   - 硬件配置方案
   - 软件支持清单
   - 部署拓扑图

4. **实施计划** (3-5页)
   - 项目时间表 (Gantt图)
   - 里程碑和关键节点
   - 风险应对措施

### 4.3 第三册：商务方案 (5-10页)

1. **报价清单** (1-2页)
   - 设备单价
   - 总价
   - 成本分解

2. **付款计划** (1页)
   - 预付款: 30% (750万)
   - 交付时: 60% (1,500万)
   - 质保期后: 10% (250万)

3. **售后服务承诺** (2-3页)
   - 3年保修内容
   - 技术支持响应时间 (≤4小时)
   - 现场支持团队配置

4. **价值增值** (1-2页)
   - 免费培训
   - 优化建议
   - 升级保证
