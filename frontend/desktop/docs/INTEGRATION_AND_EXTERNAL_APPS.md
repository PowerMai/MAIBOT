# 与系统及其他应用联动说明

## 已实现的信息交互渠道

| 渠道 | 说明 | 使用方式 |
|------|------|----------|
| **剪贴板** | 从系统剪贴板读取文本并填入对话 | 编辑器初始页按 **⌘⇧V**（Mac）/ Ctrl+Shift+V（Windows），或复制后在输入框 ⌘V 粘贴 |
| **命令面板** | 全局快捷键唤起，统一入口 | ⌘K（Mac）/ Ctrl+K（Windows） |
| **Electron 菜单** | 与桌面系统集成 | 新建对话(⌘N)、打开文件夹(⌘O)、保存(⌘S) 等 |
| **对话/任务统一** | 最近对话与聊天区同源 | 初始页「最近对话」= 真实线程列表，点击继续即切换至该对话 |

## 与微信等外部应用的轻量联动

- **微信 / 钉钉 / 浏览器等**：应用无法直接读取其他 App 的聊天内容（系统安全与 API 限制）。
- **推荐方式**：在微信等应用中「复制」需要处理的文字，回到本应用后按 **⌘⇧V**（初始页）或对话输入框 **⌘V 粘贴**，即可把内容交给 AI 处理。
- 后续若需更强联动，可考虑：
  - **系统分享（Share Extension）**：在其他 App 中通过「分享」→「用 CCB 打开」传入内容；
  - **URL Scheme**：如 `ccb://task?text=...` 由浏览器或快捷指令唤起并传入参数。

## 可扩展的 macOS / 系统级能力

| 能力 | 说明 | 实现难度 |
|------|------|----------|
| 系统分享菜单 | 在 Safari/备忘录等中分享到本应用 | 中（需注册 Share Extension） |
| URL Scheme | 通过链接或快捷指令打开应用并传参 | 低（Electron 已支持 protocol） |
| 通知中心 | 任务完成或需确认时推送系统通知 | 低（Node `notification`） |
| 全局快捷键 | 从任意 App 唤起本应用或「从剪贴板新建」 | 低（Electron globalShortcut） |

以上扩展可在产品确认需求后按优先级接入，当前以剪贴板 + 命令面板 + 真实对话列表为主渠道，保证与业务场景一致、数据真实可用。
