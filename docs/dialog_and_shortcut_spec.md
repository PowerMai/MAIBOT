# 弹窗与快捷键规范

本文档约定全局弹窗（Dialog/Modal）的交互与快捷键行为，以及主要窗口的当前实现与目标行为。

## 一、通用规范

### 1.1 弹窗交互

| 规范项 | 要求 |
|--------|------|
| **Escape 关闭** | 所有 Modal/Dialog 支持按 Escape 关闭；关闭时清理临时状态（如待上传列表、回调 ref）。 |
| **焦点陷阱** | 打开时焦点进入弹窗内，Tab 在弹窗内循环，不跳出到背后内容。 |
| **焦点回退** | 关闭时焦点回退到触发打开的元素（若仍挂载）。 |
| **主/次按钮** | 明确主按钮（确定/提交）与次按钮（取消）；主按钮与回车绑定时需确保当前无多行输入获得焦点。 |
| **与 Composer/Thread 联动** | 例如「选择工作区文件」确定后仅关闭弹窗；空态可提供「从本地上传」等快捷入口并关闭当前弹窗。 |

### 1.2 快捷键

| 场景 | 约定 |
|------|------|
| **发送消息** | Composer 内 Cmd+Enter（或 Enter，以产品为准）触发发送；与 assistant-ui / 运行时一致。 |
| **弹窗内** | Enter 确认当前焦点按钮或列表选中项；Escape 取消/关闭。 |
| **全局** | 各功能快捷键见设置页或快捷键帮助面板，不在此重复。 |

## 二、主要窗口清单

### 2.1 工作区文件选择（Composer 附件）

- **位置**：FullEditorV2Enhanced，由 `OPEN_WORKSPACE_FILE_PICKER` 触发。
- **当前行为**：标题「选择要加入对话的工作区文件」；仅列出当前已打开的文件；空态提供「从本地上传」按钮（派发 `TRIGGER_COMPOSER_FILE_UPLOAD` 并关闭）；列表支持键盘导航（ArrowDown/ArrowUp 高亮、Enter 选中并关闭）；Escape/点击遮罩关闭由 Radix Dialog 提供。
- **目标**：已落地（含列表键盘导航）。

### 2.2 添加文件夹（Composer 附件）

- **位置**：FullEditorV2Enhanced 监听 `OPEN_FOLDER_PICKER`，Electron 下调用系统目录选择，无独立弹窗。
- **当前行为**：无弹窗；非 Electron 时 toast 提示「当前环境不支持选择文件夹…」。
- **目标**：已满足；非 Electron 降级提示已存在。

### 2.3 知识库上传文档

- **位置**：KnowledgeBasePanel，上传文档到当前选中目录。
- **当前行为**：选择文件后进入待上传列表（名称、大小、可移除单项）；「确定上传」执行上传并关闭；「取消」或 Escape 关闭并清空待上传列表；DialogDescription 明确「上传到: xxx」。
- **目标**：已落地（确定/取消、多文件列表、Escape 关闭）。

### 2.4 设置页

- **位置**：独立视图/路由，非单一弹窗。
- **当前行为**：由产品与路由决定。
- **目标**：若存在子弹窗（如某设置项弹窗），遵循上述 Escape、焦点、主/次按钮规范。

### 2.5 任务详情

- **位置**：任务相关面板/侧栏内的详情视图或弹窗。
- **当前行为**：由现有实现决定。
- **目标**：若有 Dialog，Escape 关闭、确定/取消语义明确；与「查看执行日志」等联动按现有事件契约（如 OPEN_SETTINGS）处理。

### 2.6 新建目录 / 删除确认 / 技能删除 / 生成技能草稿 等

- **位置**：KnowledgeBasePanel、其他面板。
- **当前行为**：多为 Dialog + DialogFooter 确定/取消。
- **目标**：统一 Escape 关闭（Radix 默认）、主按钮与取消按钮区分清晰；新增弹窗时按本规范实现。

## 三、落地检查与补充优化

- 工作区文件选择弹窗：已按规范优化文案、空态「从本地上传」、DialogDescription；列表支持键盘高亮 + Enter 选中。
- 知识库上传弹窗：已按规范增加待上传列表、确定/取消、Escape 关闭时清空列表。
- 文件树右键「添加到对话上下文」：文件/文件夹加入上下文后均有 Toast 反馈（composer.addedFileToContext / composer.addedFolderToContext）。
- Composer：输入框下方有 @/ 提示；工具栏提供 @、/ 快捷按钮，点击在光标处插入并聚焦输入框；输入框 onSelect 同步 cursorPosition。
- Markdown 消息区：表格使用 prose-table:border-collapse；标题/列表/代码块等 prose 样式已补全。
- 其他弹窗：在迭代时对照本节与 §1 逐项检查即可。
