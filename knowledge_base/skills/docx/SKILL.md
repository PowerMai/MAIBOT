# DOCX Skill

## 适用场景
- 生成正式报告、方案书、制度文档。
- 将结构化数据渲染为可交付 Word 文件。

## 执行步骤
1. 明确模板与章节结构。
2. 组织正文、表格、附录与元信息。
3. 生成 DOCX 并检查样式一致性。
4. 输出文件路径与交付摘要。

## 质量门
- 标题层级、编号、目录规则一致。
- 表格与引用信息完整，不留占位符。
- 输出可直接交付，无明显排版错误。

详细样式规范见：`knowledge_base/skills/document-reference.md`
