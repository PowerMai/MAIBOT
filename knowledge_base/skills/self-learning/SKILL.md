# Self Learning

## 目标
- 沉淀成功模式与失败教训。
- 让后续任务可复用历史策略与验证路径。

## 执行步骤
1. 采集任务输入、输出、结果状态和关键决策。
2. 提取可迁移模式（workflow、risk、verification）。
3. 写入学习存储并更新置信度。
4. 对低质量模式执行衰减或清理。

## 质量门
- 记录必须可追溯到任务 ID 与证据链。
- 失败样本必须含根因与修复建议。
- 仅高质量样本进入蒸馏与推荐链路。
