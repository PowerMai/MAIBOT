# Document Skills Reference

用于 `docx/pdf/pptx/xlsx` 的详细规范补充。

## 通用要求
- 输入口径一致、来源可追溯、输出文件可直接交付。
- 所有占位符必须在交付前清理。
- 保留生成摘要：输入、处理步骤、输出路径、风险说明。

## DOCX
- 统一标题层级与编号。
- 长文档包含目录和版本信息。

## PDF
- 解析结果需保留页码与位置索引。
- 导出时检查编码、分页与字体。

## PPTX
- 叙事链建议：背景 -> 问题 -> 方案 -> 价值 -> 行动。
- 每页一个核心信息，图表优先结论导向。

## XLSX
- 关键公式抽样复算，保留异常数据标签。
- 数据口径与单位在首个说明页固定声明。
