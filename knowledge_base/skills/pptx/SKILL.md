# PPTX Skill

## 适用场景
- 方案汇报、业务复盘、技术宣讲。
- 需要“结构化叙事 + 图表表达”的演示交付。

## 执行步骤
1. 定义受众、目标与演示主线。
2. 规划页结构（封面/目录/正文/结论）。
3. 生成内容、图表与视觉层级。
4. 检查逻辑连贯与时间可讲述性。

## 质量门
- 一页一个核心信息，避免信息过载。
- 图表结论明确，数据来源可追溯。
- 结尾给行动建议与下一步计划。

## 产出路径
- 生成文件须写入工作区的 output 目录（或由调用方传入的 output_dir）；脚本的输入/输出路径由调用方通过参数传入，勿写死为 `data/` 或项目根。
