# 关键术语和定义

**领域**: 招投标 / 采购  
**用途**: 供 Agent 与 search_knowledge 检索使用

---

## 1. 招标与投标

| 术语 | 定义 |
|------|------|
| 招标人 | 提出招标项目、进行招标的法人或组织（采购方） |
| 投标人 | 响应招标、参加投标竞争的法人或组织（供应商） |
| 招标文件 | 招标人编制的、规定采购需求、投标要求、评标办法等的文件集合 |
| 投标文件 | 投标人按招标文件要求编制的、用于响应的全部文件 |
| 投标截止时间 | 投标文件递交的截止时刻，逾期视为无效 |

---

## 2. 评分与评标

| 术语 | 定义 |
|------|------|
| 综合评估法 | 技术分、商务分、价格分等加权汇总，按总分排序的评标方法 |
| 经评审的最低投标价法 | 满足资格与符合性后，按投标报价由低到高排序的评标方法 |
| 技术分 | 对技术方案、人员、业绩等打分的分值 |
| 商务分 | 对服务、实力、履约等打分的分值 |
| 价格分 | 按招标文件规定的价格分公式计算得出的分值 |
| 基准价 | 价格分公式中用于比较的参考价格（如有效报价平均值、下浮价等） |
| 有效报价 | 符合资格与符合性、未超控制价的投标报价，用于计算基准价与价格分 |

---

## 3. 资格与符合性

| 术语 | 定义 |
|------|------|
| 资格要求 | 投标人须具备的资质、业绩、人员、证书等强制性条件 |
| 符合性 | 投标文件对招标文件实质性要求的响应程度；不符合可导致废标 |
| 废标 | 投标因严重不符合招标要求而被否决，不进入评标 |
| 强制条款/星号条款 | 招标文件中标注为必须满足、否则废标的条款 |

---

## 4. 投标文件结构常用语

| 术语 | 定义 |
|------|------|
| 技术方案/技术部分 | 技术方案说明、实施方案、质量与保障等，对应技术分 |
| 商务部分 | 报价、服务承诺、商务条款响应等，对应商务分与价格分 |
| 偏离表/偏离说明 | 对招标要求逐条响应，标明“符合”“正偏离”“负偏离”及说明 |
| 正偏离 | 投标优于招标要求 |
| 负偏离 | 投标低于招标要求，可能影响得分或导致废标 |
